# TestLeds

## Description

A simple test utility that cycles the colors of the on-board LEDs (LED_1 and LED_2) through an RGB rainbow pattern using a sine-wave based hue shifter. The speed of the color cycle is tied to the main loop's update rate (10ms). Audio is passed through from input to output without modification.
