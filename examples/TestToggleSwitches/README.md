# TestToggleSwitches

## Description

A test utility to verify the functionality of all 6 three-position toggle switches. Toggle Switches 1-3 control the Red, Green, and Blue intensities of LED 1, while Toggle Switches 4-6 control LED 2. Audio is passed through without modification.
