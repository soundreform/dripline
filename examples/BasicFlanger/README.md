# BasicFlanger

## Description

A classic flanging effect that uses a modulated delay line to create sweeping comb-filter sounds.

Ported from the [DaisyExamples](https://github.com/electro-smith/DaisyExamples) example originally written by Electro-Smith.

### Controls

| CONTROL      | DESCRIPTION | COMMENTS                     |
| ------------ | ----------- | ---------------------------- |
| KNOB_1       | Mix         | Dry to Wet mix               |
| KNOB_2       | LFO Rate    | Speed of the sweep           |
| KNOB_3       | LFO Depth   | Width of the sweep           |
| KNOB_4       | Delay       | Offset delay time            |
| KNOB_5       | Feedback    | Resonance intensity          |
| FOOTSWITCH_1 | Bypass      | Toggles the hardware relay   |
| LED_1        | Status      | Green: Active, Red: Bypassed |
