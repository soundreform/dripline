# BasicChorus

## Description

A standard chorus effect providing lush modulation and stereo widening.

Ported from the [DaisyExamples](https://github.com/electro-smith/DaisyExamples) example originally written by Electro-Smith.

### Controls

| CONTROL      | DESCRIPTION | COMMENTS                   |
| ------------ | ----------- | -------------------------- |
| KNOB_1       | Mix         | Dry to Wet                 |
| KNOB_2       | LFO Rate    | Speed of the modulation    |
| KNOB_3       | LFO Depth   | Depth of the modulation    |
| KNOB_4       | Delay       | Base delay time            |
| KNOB_5       | Feedback    | Amount of signal fed back  |
| FOOTSWITCH_1 | Bypass      | Toggles the hardware relay |
| LED_1        | Status      | Green: Active, Red: Bypass |
