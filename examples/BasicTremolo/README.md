# BasicTremolo

## Description

A standard amplitude modulation effect featuring multiple LFO waveforms.

Ported from the [DaisyExamples](https://github.com/electro-smith/DaisyExamples) example originally written by Electro-Smith.

### Controls

| CONTROL      | DESCRIPTION | COMMENTS                                             |
| ------------ | ----------- | ---------------------------------------------------- |
| KNOB_1       | Freq        | Speed of the modulation                              |
| KNOB_2       | Depth       | Intensity of the volume dip                          |
| KNOB_3       | Waveform    | Selects LFO shape (Sine, Triangle, Saw, Square, etc) |
| FOOTSWITCH_1 | Bypass      | Toggles the hardware relay                           |
| LED_1        | Status      | Green: Active, Red: Bypassed                         |
