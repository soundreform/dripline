# BasicPhaser

## Description

A multi-stage phase shifter that creates swirling, psychedelic modulation by sweeping all-pass filters across the frequency spectrum.

Ported from the [DaisyExamples](https://github.com/electro-smith/DaisyExamples) example originally written by Electro-Smith.

### Controls

| CONTROL      | DESCRIPTION | COMMENTS                      |
| ------------ | ----------- | ----------------------------- |
| KNOB_1       | Mix         | Dry to Wet mix                |
| KNOB_2       | LFO Rate    | Swirl speed                   |
| KNOB_3       | LFO Depth   | Intensity of modulation       |
| KNOB_4       | Frequency   | Base filter frequency         |
| KNOB_5       | Feedback    | Resonance depth               |
| KNOB_6       | Stages      | Number of phaser stages (1-8) |
| FOOTSWITCH_1 | Bypass      | Toggles the hardware relay    |
| LED_1        | Status      | Green: Active, Red: Bypassed  |
