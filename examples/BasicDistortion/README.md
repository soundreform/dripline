# BasicDistortion

## Description

A dual-stage distortion effect featuring soft and hard clipping stages that can be toggled independently.

Ported from the [DaisyExamples](https://github.com/electro-smith/DaisyExamples) example originally written by Electro-Smith.

### Controls

| CONTROL      | DESCRIPTION | COMMENTS                          |
| ------------ | ----------- | --------------------------------- |
| KNOB_1       | Mix         | Dry to Wet mix                    |
| KNOB_2       | Pregain     | Input boost before Gain           |
| KNOB_3       | Gain        | Clipping intensity                |
| FOOTSWITCH_1 | Soft Clip   | Toggles Soft Clipping stage       |
| FOOTSWITCH_2 | Hard Clip   | Toggles Hard Clipping stage       |
| LED_1        | Soft Status | Blue when Soft Clipping is Active |
| LED_2        | Hard Status | Red when Hard Clipping is Active  |
